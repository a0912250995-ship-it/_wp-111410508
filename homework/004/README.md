# JavaScript 基礎語法練習題庫

> **AI 輔助工具使用聲明**：本專案在開發過程中使用了 opencode (AI 程式設計助手) 作為輔助工具，用於除錯、語法諮詢、概念釐清與程式碼生成。

本專案收錄了 10 個精選的 JavaScript 程式練習題，旨在幫助初學者透過實作掌握程式核心邏輯。

## 🎯 涵蓋的核心觀念
本題庫的設計包含並整合了以下關鍵技術點：
* **條件控制** (`if...else`)
* **迴圈控制** (`for`、`while`)
* **函式封裝** (`function`)
* **資料結構** (陣列 `array`、物件 `object`)
* **資料轉換** (`JSON` 解析與字串化)

## 🛠️ 如何在本地端執行此練習？

您可以使用 **Node.js** 或直接透過 **瀏覽器開發者工具** 來執行這些程式碼。

### 方法一：使用 Node.js（推薦）
1. 請確認您的電腦已安裝 [Node.js](https://nodejs.org/)。
2. 在您的專案資料夾中建立一個 JavaScript 檔案（例如 `practice.js`）。
3. 將練習題程式碼貼入檔案中。
4. 打開終端機（Terminal）並輸入以下指令執行：
   ```bash
   node practice.js
